# Mushee Yield — Powered by Starkzap

**Mushee (UK incorporated)**  
Tagline: **Make yield feel like a button.**

Mushee Yield is a simple staking dashboard built with the **Starkzap SDK**.
The goal is one clean loop: **connect → pick validator → load pools → stake → track → claim**.

## What’s included
- Starkzap onboarding (Cartridge strategy)
- Validator presets + pool discovery
- One-tap staking flow
- Dashboard with position refresh
- Best-effort balance fetch + “Max” UX
- Metrics panel (auto-detect APR/APY fields when present)

## Run locally
```bash
npm install
npm run dev
```

## Deploy on Vercel
1) Push to GitHub and import to Vercel (Next.js preset)
2) Deploy
3) Open the app → Links tab → paste your Demo URL + Repo URL
